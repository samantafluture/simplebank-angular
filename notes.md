# Angular

## Preparar ambiente

- instalar a CLI `npm install -g @angular/cli`
- criar o projeto `ng new my-first-project`
- ir para a pasta do projeto `cd my-first-project`
- rodar o projeto `ng serve`
- acessar `http://localhost:4200/`
